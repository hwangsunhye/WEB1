      HTML elements are the building blocks of HTML pages. With HTML constructs, images and other object, such as interactive froms, may be embedded into the rendered page. it provides a means to create structured documents by denoting structural semantis for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
